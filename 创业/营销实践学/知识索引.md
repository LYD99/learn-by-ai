```
【第1层】钩子：戳痛点（1句话）
【第2层】稀缺：制造FOMO（2句话）
【第3层】背书：数据+故事+信任（2句话）
【第4层】行动：一步到位的指令（1句话）
```

#### 标题三层（第二课）
```
【第1层】钩子词汇：激发好奇或戳痛点（1-3词）
【第2层】数字信息：加入具体数字和对比（数字 + 时间/对比）
【第3层】行动诱惑：暗示"不动就错过"（晚到涨价、仅X个、截止XX）
```
